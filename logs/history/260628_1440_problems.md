# Problem Record (Troubleshooting)

**Severity**: Medium

## The Problem

Durante la auditoria posterior de `DG-PSEA-02` y `DG-PSEA-03`, se encontro que habian reaparecido elementos no uniformes con el estilo documental del SGC:

- Metadatos de encabezado como `Codigo`, `Tipo de documento`, `Fecha de actualizacion`, `Version documental` y equivalentes.
- Una seccion operativa en `DG-PSEA-02` llamada `Registro de preparacion de este documento`.
- Un archivo tecnico `indice_capturas.md` dentro de la carpeta de `DG-PSEA-03`.
- Carpetas con nombres tecnicos `dgpsea02` y `dgpsea03` que no eran suficientemente documentales.
- Rutas canonicas antiguas en inventario y matriz documental que apuntaban a archivos Markdown en la raiz de `01_documentos_marco`.

## User-Requested Corrections

- Mantener las carpetas, pero ajustarles los nombres.
- Quitar `indice_capturas.md`.
- Actualizar mapa e inventario.
- Aplicar los ajustes identificados en la auditoria previa.

## The Solution / Workaround

- Se renombraron las carpetas a:
  - `DG-PSEA-02 Aplicativo calaire-app`
  - `DG-PSEA-03 Aplicativo pt_app`
- Se elimino `indice_capturas.md`.
- Se limpiaron encabezados y secciones no uniformes en los Markdown.
- Se actualizaron referencias internas a `capturas`.
- Se actualizo `tomar_capturas.js` con la nueva ruta de `DG-PSEA-03`.
- Se actualizaron rutas en:
  - `Inventario Documental del SGC.md`
  - `P-PSEA-02 Matriz documental basica del PEA.md`
- Se actualizo `mapa_navegacion_sgc_pea.html` para reflejar estado `Elaborado` y carpetas con evidencias.
- Se regeneraron HTML y DOCX desde Markdown usando la plantilla DG base.

## Learnings to Preserve

- Las evidencias de preparacion, comandos, capturas o automatizacion pertenecen a logs o soporte tecnico, no al cuerpo del documento general controlado.
- Si un DG necesita imagenes, pueden vivir en una subcarpeta `capturas/`, pero el documento debe mantener estilo SGC uniforme.
- Al mover documentos controlados a carpetas propias, actualizar inmediatamente inventario, matriz documental y cualquier mapa de navegacion.
- Ejecutar Pandoc desde la carpeta del Markdown cuando las imagenes usan rutas relativas como `capturas/...`; si se ejecuta desde otra ubicacion, Pandoc puede reemplazar imagenes por descripciones.
