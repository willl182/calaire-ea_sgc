# Carpeta `para_quitar`

Esta carpeta contiene documentos Markdown del SGC PEA que **no figuran en el mapa de navegación documental vigente** (`docs/documentacion_sgc/mapa_navegacion_sgc_pea.html`) ni en el índice de fichas resumen (`docs/documentacion_sgc/fichas_resumen/ficha_names.js`).

## Criterio de movimiento

Se movieron aquí los 23 archivos cuyo código documental (`DG-PSEA`, `P-PSEA`, `I-PSEA`, `F-PSEA`) no aparece en `ficha_names.js`. Estos documentos corresponden a versiones preliminares, borradores antiguos o códigos que quedaron fuera del alcance documental estabilizado del PEA.

Los documentos se conservan por si su contenido resulta útil para futuras fichas o para trazabilidad histórica. Si alguno se reincorpora al mapa documental, debe:

1. Revisar que su código y nombre coincidan con `ficha_names.js`.
2. Moverlo a la ubicación activa correspondiente en `docs/documentacion_sgc/`.
3. Actualizar `mapa_navegacion_sgc_pea.html` y/o `fichas_resumen/` si es necesario.
